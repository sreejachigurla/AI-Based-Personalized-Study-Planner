# 📚 AI-Based Personalized Study Planner

An intelligent study planning web application that uses **Machine Learning** to analyze student data and generate **personalized study schedules and recommendations**.

---

## 🚀 Features

* 📊 Analyzes student performance data
* 🤖 Recommends subjects/topics based on strengths & weaknesses
* 🗓️ Generates personalized study schedules
* 📈 Tracks progress using CSV data
* 🔍 Supports data visualization and analytics

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Framework:** Flask
* **Libraries:** Pandas, Scikit-learn, Matplotlib
* **Data Handling:** CSV Files
* **Concepts:** Machine Learning, Recommendation Systems, Data Analysis

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/sreejachigurla/AI-Based-Personalized-Study-Planner.git
cd AI-Based-Personalized-Study-Planner
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the application

```
python app.py
```

---

## 🔑 Environment Variables

Create a `.env` file (if using API keys):

```
API_KEY=your_api_key_here
```
